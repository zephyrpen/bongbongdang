# 🍵 봉봉당 (BongBongDang)

따뜻하고 정성스러운 수제 쌍화차 판매 웹앱입니다.

## ✨ 주요 기능

- **회원 전용 주문**: 회원가입 및 로그인한 사용자만 주문 가능
- **실시간 금액 계산**: 수량에 따른 상품 금액 및 배송비(3개 이상 무료) 자동 계산
- **반응형 디자인**: PC 및 모바일 최적화
- **감성적인 UI**: 따뜻한 한방 감성의 디자인과 애니메이션

## 🛠 기술 스택

- **Framework**: React + Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Font**: Nanum Myeongjo (Serif), Noto Sans KR (Sans)

## 🚀 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## ⚠️ 주의사항

- 본 프로젝트는 데모/포트폴리오용으로 제작되었습니다.
- 결제 시스템은 연동되어 있지 않으며, 계좌이체 안내만 표시됩니다.
- 회원 정보는 브라우저의 LocalStorage에 저장됩니다.
